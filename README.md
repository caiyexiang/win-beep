# win-beep

[![NPM version][npm-image]][npm-url]

Windows-only native addon to play back a beep to user.

## Installing

```
npm install win-beep --save-dev
```

## Usage

```js
const { beep } = require('win-beep')
beep(600, 1000) // frequency: 600Hz, duration: 1000ms
```
